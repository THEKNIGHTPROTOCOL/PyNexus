This is a simple alarm clock application built with Python and Tkinter. It lets you set an alarm, shows the current time, and plays a sound when the alarm goes off!

Features ‚ú®

  • Easy-to-use Interface: A friendly graphical interface made with Tkinter.
  • Real-time Clock: Always displays the current time (hour, minute, second).
  • Set Your Alarm: Pick your desired hour, minute, and second using dropdowns.
  • Sound Alert: Plays a sound.wav file when the alarm time hits (you can add your own!).
  • Runs Smoothly: The alarm checking works in the background so the app stays responsive.

How to Use üöÄ

  1. Save the code: Save the provided Python code as alarm_clock.py.
  2. Add a sound (optional): If you want an audible alarm, put a .wav sound file named sound.wav in the same folder as your script. If you don't have one, the alarm will still show a message!
  3. Run it: Open your terminal or command prompt, navigate to the folder where you saved the file, and run: 
    python alarm_clock.py
  4. Set the time: Use the dropdown menus to choose the hour, minute, and second for your alarm.
  5. Click "Set Alarm": Hit the "üîî Set Alarm" button to activate it.
  6. That's it!: The alarm will pop up a message and play your sound when the time comes.
